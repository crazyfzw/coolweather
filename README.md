# coolweather
酷欧天气

学习郭霖大神的《第一行代码》后，练手的项目实战。


酷欧天气是一款基于Android端开源的天气预报软件，具备查看全国的省市县、查询任意城市天气、自由切换城市、手动更新天气、后台自动更新天气等功能。酷欧天气中的数据均由中国气象局提供，代码遵循Apache v2 License开源协议。本软件主要作为学习和交流使用。
